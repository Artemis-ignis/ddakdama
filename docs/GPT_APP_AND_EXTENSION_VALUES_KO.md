# GPT 앱·확장 프로그램 최종 연결값

## 결론

일반 사용자는 서버 주소, API 키 또는 토큰을 입력하지 않습니다. 운영자가 ChatGPT에 앱을 한 번 등록하면 사용자는 딱담아 확장 프로그램의 `연결하기` 버튼으로 6자리 코드를 만든 뒤 GPT 앱에 입력하기만 하면 됩니다.

OpenAI API 키는 이 연결에 필요하지 않으며, 목록 파싱·페어링·확장 프로그램 전달 과정에서 별도 OpenAI API 사용료가 발생하지 않습니다.

## 확장 프로그램 설치값

Chrome의 `압축해제된 확장 프로그램을 로드합니다`에서 다음 폴더 하나만 선택합니다.

`C:\Users\50106\Desktop\딱담아\apps\extension`

별도의 서버 주소, MCP URL, OpenAI API 키 또는 연결 토큰은 입력하지 않습니다. 공개 서버 주소는 빌드에 포함되어 있습니다.

## 실제 연결 순서

1. Chrome에서 딱담아 사이드 패널을 엽니다.
2. `ChatGPT에서 목록 받기` 영역의 `6자리 코드 만들기`를 누릅니다.
3. ChatGPT 딱담아 앱 위젯의 `확장 프로그램 연결`에 코드를 입력합니다.
4. 위젯에서 `연결`을 누르고 `연결됨` 표시를 확인합니다. 확장 프로그램은 코드 발급만으로 연결됐다고 표시하지 않으며 서버의 실제 연결 완료 상태를 확인합니다.
5. 연결이 완료되면 GPT 앱에서 `확장 프로그램으로 보내기`를 누릅니다.
6. 확장 프로그램에서 `목록 받기`를 누릅니다.
7. 실제 상품·가격·수량을 확인하고 최종 담기를 승인합니다.

연결을 해제했거나 코드가 만료되면 확장 프로그램에서 `연결하기` 또는 `새 코드`를 눌러 새 6자리 코드를 만들면 됩니다.

## 데모 영상 공개 범위

Devpost나 앱 심사자가 봐야 한다면 YouTube의 완전 비공개보다 `일부 공개(링크가 있는 사용자만)`가 적합합니다. 완전 비공개 영상은 초대된 Google 계정 외에는 재생할 수 없습니다.
