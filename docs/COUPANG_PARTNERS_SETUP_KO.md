# 쿠팡 파트너스 설정

- Access Key와 Secret Key는 서버 secret에만 저장합니다.
- 확장 프로그램이나 ChatGPT 위젯에 키를 넣지 않습니다.
- 파트너스 공식 로그인 화면에서 현재 API 권한과 규격을 확인한 뒤 연결합니다.
- API에서 검색되지 않는 상품은 일반 쿠팡 브라우저 검색으로 처리합니다.
- Deep Link 생성 실패는 전체 검색이나 장바구니 계획을 중단시키지 않습니다.
- 링크 생성은 수수료를 보장하지 않으며 취소·반품·본인 구매 등의 판정은 쿠팡 정책을 따릅니다.
- Chrome Web Store 공개판은 직접 사용자 혜택이 실제로 구현되지 않은 동안 제휴 삽입을 비활성화해야 합니다.
